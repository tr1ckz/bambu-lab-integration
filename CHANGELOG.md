# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-03

### Added
- Initial release
- OIDC authentication support (Authentik, Keycloak, Auth0, etc.)
- Print history tracking with MakerWorld cover images
- Model library management (.3mf, .stl, .gcode)
- Real-time printer monitoring via MQTT
- Automatic cloud synchronization with Bambu Lab
- Timelapse video download and conversion
- Print statistics and analytics
- Duplicate model detection
- Multi-user support with admin controls
- Docker support with multi-architecture builds (amd64, arm64)
- Unraid deployment support
- Buy Me a Coffee integration for developer support
- React 19.2 frontend with TypeScript
- Express.js backend with SQLite database
- Loading screens with animations
- Styled confirmation modals
- Responsive sidebar navigation

### Documentation
- Comprehensive README with quick start guide
- Environment configuration guide (README-ENV.md)
- Docker deployment instructions (README-DOCKER.md)
- Unraid installation guide (UNRAID-INSTALL.md)
- Code cleanup documentation (CLEANUP.md)
- Contributing guidelines (CONTRIBUTING.md)
- GitHub Actions workflow for automated builds

### Infrastructure
- GitHub Actions for multi-architecture Docker builds
- Automated publishing to GitHub Container Registry
- Optional Docker Hub support
- Build caching for faster builds
- Semantic versioning support
